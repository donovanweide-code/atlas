# Atlas

Atlas is de manier waarop We Build And Design denkt, beslist en bouwt.

Begin altijd bij [Foundation.md](Foundation.md). Dit is de enige bron van waarheid voor missie, klant, methode, principes en creatieve taal.

## Begin je werkdag

1. Start de website met `npm run dev` vanuit `/website`.
2. Open `/atlas` als interne Workspace.
3. Toets de eerstvolgende stap aan het kompas: **helpt dit de ondernemer vandaag écht verder?**
4. Leg betekenisvolle beslissingen en lessen vast in het Atlas Logboek.

## Structuur

- `/website` — publieke website, Atlas Workspace en Atlas Lab.
- `/docs/atlas` — operationele Atlas-documentatie, beslissingen en sprints.
- `/clients` — casecontext en herbruikbare lessen per relatie.
- `/Creative-Direction` — de eigen visuele taal en ontwerpgeschiedenis van Atlas.
- `/playground` — experimenten die nog geen onderdeel van de praktijk zijn.

Software laat zien wat we gebouwd hebben. Atlas bewaart waarom.
