# Atlas Visual Language

Deze map bevat de primaire visuele referentie van Project Atlas.

De Visual Language staat tussen filosofie en uitvoering:

1. Atlas Philosophy beschrijft waarom Atlas bestaat.
2. Atlas World System beschrijft hoe de wereld werkt.
3. Atlas Visual Language laat zien hoe die wereld eruitziet en voelt.
4. Specialistische vision boards verdiepen afzonderlijke onderwerpen.
5. Creative Directions vertalen dit naar concrete ervaringen.

Iedere nieuwe visuele ontwerpbeslissing wordt getoetst aan de actieve versie van de Atlas Visual Language.

De actieve afbeelding krijgt de naam:

`Atlas-Visual-Language-v01.png`

Oude versies worden niet overschreven en blijven als historisch referentiemateriaal bewaard.
