# 03 — Navigation

## Doel

Dit board bepaalt hoe bezoekers richting ontdekken zonder een kaartinterface uitgelegd te krijgen.

## Gewenste emotie

Oriëntatie, vertrouwen, nieuwsgierigheid en rustige voortgang.

## Visuele kenmerken

Organische routes, waypoints, topografie, subtiele coördinaten, sterren, kruispunten en ingetogen kompasreferenties.

## Wat past

- één doorlopende route;
- betekenisvolle waypoints;
- markeringen die pas bij aandacht zichtbaar worden;
- natuurlijke en onregelmatige lijnen.

## Wat niet past

- standaard kaartinterfaces;
- GPS-styling;
- futuristische HUD-elementen;
- bewegende streepjes;
- willekeurige iconen.

## Gebruik door Codex

Gebruik navigatie om de volgende emotionele stap voelbaar te maken. Voeg alleen een waypoint of route toe wanneer duidelijk is wat het in het verhaal betekent.
