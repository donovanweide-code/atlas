# 001 - Wat is Git?

## Wat is Git?

Git is een versiebeheersysteem.

Het bewaart de volledige geschiedenis van een project, zodat je altijd kunt teruggaan naar een eerdere versie.

---

## Waarom gebruiken we Git?

- Veilig werken
- Altijd een back-up van wijzigingen
- Samenwerken
- Experimenteren zonder risico
- Professionele workflow

---

## Belangrijkste commando's

### git init

Start een Git-project.

### git status

Laat zien welke bestanden gewijzigd zijn.

### git add .

Voegt alle wijzigingen toe aan de volgende commit.

### git commit -m "Omschrijving"

Slaat een nieuwe versie van het project op.

### git push

Stuurt de nieuwe versie naar GitHub.

---

Vandaag heb ik geleerd hoe ik:

- Mijn eerste commit maak
- Mijn project naar GitHub stuur
