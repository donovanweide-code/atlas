# Atlas Logboek

Het Logboek bewaart betekenis, niet iedere activiteit. Voeg een notitie toe wanneer een beslissing, les, succes of verandering in momentum later opnieuw waarde kan leveren.

## Vast format

### YYYY-MM-DD — Korte titel

- **Type:** beslissing / les / succes / momentum
- **Case:** nummer en naam
- **Context:** wat gebeurde er?
- **Betekenis:** waarom is dit relevant?
- **Volgende stap:** wat verdient nu aandacht?
- **Herbruikbare les:** wat mag een volgende case hiervan meenemen?
